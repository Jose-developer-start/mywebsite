HOST virtuales en Apache2

Autor: José Deodanes

Programador

Publicado: 18/01/2022

1. Paquetes requeridos

Instalaremos el paquete Apache2 con el siguiente comando:

# apt install apache2

2. Configuración

Agregar las siguiente líneas en el archivo /etc/apache2/apache2.conf


                        <Directory /var/www/tu_dominio.com/public/>
                            Options Indexes FollowSymLinks
                            AllowOverride All
                            Require all granted
                        </Directory>

Para crear un Apache virtual host, vamos a utilizar el editor nano de la siguiente forma:

# nano /etc/apache2/sites-available/tu_dominio.com.conf 

                        <VirtualHost *:80>
                            ServerName tu_dominio.com
                            ServerAlias www.tu_dominio.com
                            ServerAdmin webmaster@localhost
                            DocumentRoot /var/www/tu_dominio.com/public/

                            ErrorLog ${APACHE_LOG_DIR}/error.log
                            CustomLog ${APACHE_LOG_DIR}/access.log combined
                        </VirtualHost>

Ya casi terminamos, lo único que queda por hacer es registrar el nuevo vhost. Para esto ejecutamos el siguiente comando:

# sudo a2ensite tu_dominio.com.conf

Y, reiniciamos nuestro servidor web Apache:

# sudo service apache2 reload

Opcional. Habilitar el módulo mod_rewrite de Apache

Este paso es importante para que funcionen correctamente nuestras rutas de Laravel. Ejecutamos el siguiente comando:

# sudo a2enmod rewrite
