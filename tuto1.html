Aprende a instalar Let's Encrypt SSL

Autor: José Deodanes

Programador

Publicado: 18/01/2022

1. Paquetes requeridos

Primero instalaremos los siguientes comandos:

#apt install software-properties-common -y

Ahora, instalaremos el programa Certbot, el cual es un demonio que aplica el certificado SSL a tu dominio, pàra instalar este programa, lo haremos con el siguiente comando

#apt install certbot -y

Listo, ahora instalaremos el paquete de CertBot para Apache

#apt install python3-certbot-apache

Listo, ya tenemos CertBot instalado en nuestro servidor.

2. Instalaremos el certificado Let's Encrypt

Use el comando certbot para crear un certificado Let's Encrypt y configure Apache para usar el certificado.

#certbot --apache

Ahora, se ejecuta un shells y seguiras estos pasos:

1. (Enter 'c' to cancel): Tu correo electrónico << Enter Email ID

2. agree in order to register with the ACME server. Do you agree? Y

3. (Y)es/(N)o: N << Subscriber to Newsletter

Seleccione el sitio para instalar el certificado SSL de Let's Encrypt.

Listo, nuestro certificado ya esta instalado

Renovar el certificado

Para la renovación automática del certificado SSL debes de ejecutar el siguiente comando

# certbot renew --force-renewal

Si genera error, usa este otro:

# certbot renew --dry-run

Instalar el certificado del golpe

Bien, el siguiente comando instala de forma automática el certificado, de esta manera te ahorras de ir seleccionando de forma manual, para esto, usa el siguiente comando:

#certbot certonly --standalone --preferred-challenges http --agree-tos --email tu_correo@outlook.com -d tu_dominio.com
